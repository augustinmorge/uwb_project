INS configuration log

This document is produced by processing the configuration telegrams embedded in a post-processing file recorded from an iXBlue INS.

It displays the configuration of the equipment and changes made along time during the operation. The changes are highlighted using the following color code:

  • green - new data added in the current configuration
  • blue - data changed from the previous configuration
  • red - data deleted in the current configuration
  • black - the data is identical to the previous configuration value.
