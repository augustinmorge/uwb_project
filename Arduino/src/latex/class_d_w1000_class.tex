{\footnotesize\ttfamily void DW1000\+Class\+::enable\+Mode (\begin{DoxyParamCaption}\item[{const byte}]{mode\mbox{[}$\,$\mbox{]} }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Specifies the mode of operation for the DW1000. Modes of operation are pre-\/defined combinations of data rate, pulse repetition frequency, preamble and channel settings that properly go together. If you simply want the chips to work, choosing a mode is preferred over manual configuration.

The following modes are pre-\/configured and one of them needs to be chosen\+:
\begin{DoxyItemize}
\item {\ttfamily MODE\+\_\+\+LONGDATA\+\_\+\+RANGE\+\_\+\+LOWPOWER} (basically this is 110 kb/s data rate, 16 MHz PRF and long preambles)
\item {\ttfamily MODE\+\_\+\+SHORTDATA\+\_\+\+FAST\+\_\+\+LOWPOWER} (basically this is 6.\+8 Mb/s data rate, 16 MHz PRF and short preambles)
\item {\ttfamily MODE\+\_\+\+LONGDATA\+\_\+\+FAST\+\_\+\+LOWPOWER} (basically this is 6.\+8 Mb/s data rate, 16 MHz PRF and long preambles)
\item {\ttfamily MODE\+\_\+\+SHORTDATA\+\_\+\+FAST\+\_\+\+ACCURACY} (basically this is 6.\+8 Mb/s data rate, 64 MHz PRF and short preambles)
\item {\ttfamily MODE\+\_\+\+LONGDATA\+\_\+\+FAST\+\_\+\+ACCURACY} (basically this is 6.\+8 Mb/s data rate, 64 MHz PRF and long preambles)
\item {\ttfamily MODE\+\_\+\+LONGDATA\+\_\+\+RANGE\+\_\+\+ACCURACY} (basically this is 110 kb/s data rate, 64 MHz PRF and long preambles)
\end{DoxyItemize}

Note that LOWPOWER and ACCURACY refers to the better power efficiency and improved transmission performance of 16 MHZ and 64 MHZ PRF respectively (see {\ttfamily \doxylink{class_d_w1000_class_a06b2868d23f50189b22b6a67f089e04d}{set\+Pulse\+Frequency()}}).

The default setting that is selected by {\ttfamily \doxylink{class_d_w1000_class_a07dcb51bbdd1d2628aca0aebcd4e8648}{set\+Defaults()}} is MODE\+\_\+\+LONGDATA\+\_\+\+RANGE\+\_\+\+LOWPOWER.
